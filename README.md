# Mini Hack Cohort 2 - Gaming and Gamified Solutions on Avalanche

Fork this repo. Build a game or gamified system. Deploy to Fuji testnet.

## Prerequisites

Cohort 1 completion or equivalent: Solidity basics, Hardhat, ERC-20 deployment on Fuji.

## Getting started

```bash
git clone https://github.com/YOUR-HANDLE/minihack-cohort2-template.git
cd minihack-cohort2-template
npm install
cp .env.example .env
npx hardhat compile
```

## Weekly deliverables

| Week | Deliverable |
|------|-------------|
| 1 | In-game NFT asset minted on Fuji - contract + IPFS metadata + UI display |
| 2 | Game contract with reward token + in-game shop |
| 3 | Multiplayer on-chain leaderboard + WebSocket backend |
| 4 | Final game or gamified system - deployed frontend + all contracts + demo video |

## Weekly branch format

```
week-{N}-{your-github-handle}
```
